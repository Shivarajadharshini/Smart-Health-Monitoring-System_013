# Smart Health Monitoring System

## Description

This project is a Python-based Smart Health Monitoring System that simulates health parameters such as heart rate, SpO₂, and body temperature. It uses a graphical user interface (GUI) built with Tkinter to display real-time data and alert abnormal values.

## Features

* Real-time health data simulation
* Start/Stop monitoring
* Alert system for abnormal values
* Live timestamp display

## Files

* `health_monitor.py` – basic simulation
* `health_monitor_gui.py` – GUI version
